# Base Project
Шаблонный проект для быстрого старта разработки приложения.

## Технологии
Проект разрабатывался с использованием:

- Spring Framework (Security, Core, Boot, Data JPA, HikariCP)
- PostgreSQL
- Redis
- etc. (Lombok, MapStruct, Swagger)
